Presenteprofe es una extensión de Chrome diseñada para ayudar a los profesores a tomar la asistencia de los alumnos de manera rápida y fácil.
La extensión proporciona una interfaz simple, intuitiva e interactiva para registrar la presencia o ausencia de los estudiantes en una clase.

  • Presiona las fechas ← → para cambiar de alumno
  • Presiona la fecha hacia arriba ↑ para dejarlo PRESENTE
  • Presiona la fecha hacia abajo ↓ para dejarlo AUSENTE
  • Simplificar el proceso de toma de asistencia para los profesores.
  • Reducir el tiempo y esfuerzo necesario para registrar la asistencia de los estudiantes.
  • Proporcionar informes precisos y detallados de asistencia para una fácil revisión y análisis.
  • Permitir a los profesores tomar la asistencia desde cualquier lugar con acceso a Internet.
  • Facilitar la gestión de datos de asistencia y reducir la posibilidad de errores humanos.
